# üìù TO-DO List App

![Demo](TO-DO.png)

A simple, interactive, and fully responsive **TO-DO list web application** built with **HTML, CSS, and JavaScript**.  
Manage tasks with priorities, mark them as completed, edit, delete, and filter by status. Tasks are saved in **localStorage** so your list persists across page reloads.

---

## **Features**

- Add new tasks with **Low, Medium, or High** priority  
- Mark tasks as **completed**  
- **Edit tasks** via double-click  
- **Delete tasks** with confirmation  
- Filter tasks: **All / Completed / Pending**  
- **Inline priority editing** by clicking on the priority label  
- **LocalStorage** persists tasks  
- Fully **responsive design** for desktop and mobile  
- Clean and professional UI

## **Usage**

1. Enter a task in the input field.  
2. Select priority (Low / Medium / High).  
3. Click **Add** or press **Enter**.  
4. Click the **checkbox** to mark complete.  
5. Double-click the task text to edit.  
6. Click **Delete** to remove a task.  
7. Click **priority label** to change priority inline.  
8. Use **All / Completed / Pending** filters to view tasks.
